边缘检测和轮廓
此方法基于两个概念：边缘检测和轮廓。边缘检测，顾名思义，就是试图在图像中找到对比线或边缘。这个关键的第一步是对图像进行预处理，以帮助区分任何物体。有几种边缘检测方法存在，其中，Canny方法是非常受欢迎的，并打包在OpenCV中。
